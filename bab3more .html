Android Studio

Desain Form login sederharna

a. Pertama download android studio dan install pada computer.

b. Buka android studio

c. Setelah membuka android studio,buatlah new project dengan menekan tombol “New Project”

d. Setelah menekan “New Project” akan muncul Popup untuk memilih activity yang akan di pakai,pilihlah empty activity

e. Setelah memilih activity,akan muncul Popup dimana dapat memilih nama aplikasi,nama package, bahasa pemograman yang akan di pakai,lokasi dimana file akan di simpan dan minimum SDK yang akan digunakan,dan tekan “Next”.Setelah menekan “Next” akan membuka file activity_main.xml.

f. Akan disambut dengan tampilan dari activity_main.xml seperti dibawah ini



g. Kita dapat menghapus tulisan “Hello Wolrd” pada activity_main.xml.

h. Setelah itu kita dapat menambahkan option Text untuk membuat formulir login dan gunakan “Plain Text”


i. Setelah itu kita dapat mengganti nama dari “Plain Text” menjadi “Email” dan “Password” pada code,dengan merubah “android:text=”Name” menjadi email atau password.

j. Setelah itu kita akan dapat menambahkan Button di bawah semua text,lokasi Button berada di dalam palette “Buttons” dan kita dapat mengganti warna dengan cara merubah “android:backgroundTint=”(#warna yang di inginkan) dan gantilah text menjadi login dengan cara mengganti “android:text=”Login”.
